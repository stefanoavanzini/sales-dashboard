# Sales Dashboard — Power BI

This project showcases the creation of an interactive sales dashboard using **Power BI** to analyze sales performance and monitor key business KPIs.

## Dataset
The analysis is based on the **Sample Superstore** dataset, which contains historical sales transactions including:
- Order date
- Sales and profit
- Product categories
- Regions and customer segments
- Discounts and quantities

## Dashboard Features
The Power BI dashboard includes:

### Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Number of Orders
- Average Discount

### Visual Analysis
- Sales trends over time
- Sales by region
- Profit by product category
- Interactive slicers for segment and region analysis

## Files
- `dashboard/sales_dashboard.pbix` — Power BI dashboard file
- `data/sample_superstore.csv` — dataset used for the analysis
- `screenshots/` — dashboard previews

## Purpose
This project was developed to demonstrate practical skills in:
- Business intelligence and data visualization
- KPI design and monitoring
- Interactive dashboard creation for decision support
